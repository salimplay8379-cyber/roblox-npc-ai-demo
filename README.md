# Roblox NPC AI Demo

This project demonstrates a Roblox NPC AI system with the following features:

- patrol system using patrol points
- player detection using line-of-sight
- pathfinding navigation around obstacles
- dynamic chasing behavior
- automatic jumping over small obstacles
- attack system when close to the player
- stuck detection and path recalculation

The AI uses Roblox's PathfindingService to navigate the map intelligently.

The included demo video shows the NPC patrolling, detecting the player, chasing them, navigating around walls, and jumping over obstacles.
